## 🔤 1. Королевская Типографика (Шрифты бренда)

*Если пожелает твой владыка сменить облик букв на фирменное начертание, не нужно переписывать весь свиток стилей! Наши мастера подготовили бесшовное заклинание.*

### Как влить брендовые шрифты в проект:
1. Открой врата сокровищницы по пути: `/public/fonts/`.
2. Помести туда файлы своего брендового шрифта.
3. Переименуй их строго по древнему канону:
   * `BrandSans-Regular.woff2` (или `.woff` / `.ttf`) — для обычного текста.
   * `BrandSans-Bold.woff2` (или `.woff` / `.ttf`) — для заголовков и жирных рун.
4. **Магия в деле**: Наша Прялка Стилей (`/src/index.css`) уже настроена искать эти файлы. Как только они появятся в гнезде, сайт плавно и гордо переключится на них! 
5. Если же файлов не окажется, сработает древний резервный оберег — сайт отобразится на изящном, проверенном временем шрифте **Inter**.

## 🕵️‍♂️ 2. Тайное Клеймо Создателя (Пасхалка автора)

*Каждый великий кузнец оставляет на своём мече незаметное клеймо, ведомое лишь знающим. Мастер **stsyn** наложил невидимое плетение на этот сайт, дабы подтвердить своё авторство:*

1. **Руна в Свитках Консоли**: Открой консоль разработчика в своём браузере (F12) в любой части сайта. Ты увидишь сияющее послание, начертанное при инициализации мира (`/src/components/Layout.tsx`):
   ```text
   🚀 Built & Crafted by stsyn | EVOLUTE Pragmatika Platform
   ```
2. **Незримая Печать в Подвале**: Прокрути свиток страницы до самого низа, в самые темные дебри футера (где струится юридический текст об офертах). Там, в самом конце, притаилась крошечная руна `stsyn`. 
   * Она практически невидима (`opacity-[0.05]`), сливаясь со звездной пылью фона, дабы не тревожить взор обычных смертных.
   * Но стоит навести на неё курсор — и она вспыхнет мягким светом, гордо подтверждая имя своего создателя!

## ⚙️ Алхимический Горн (Скрипты Запуска)

*Чтобы вдохнуть жизнь в проект и разжечь пламя на локальной машине, используй следующие заклинания из корня папки:*

* **`npm run dev`** — Разжечь костёр локальной разработки (на порту `3000`). Изменения в коде будут мгновенно переливаться в твой браузер!
* **`npm run build`** — Выковать чистейший, оптимизированный дистрибутив в папке `/dist` для отправки на дальние сервера.
* **`npm run lint`** — Призвать строгих духов Тайпскрипта для проверки чистоты и строгости типов без компиляции.
* **`npm run start`** — Запустить предварительный просмотр выкованного дистрибутива на порту `3000`.
